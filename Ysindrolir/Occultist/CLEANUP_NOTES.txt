Cleanup notes
============
This bundle removes duplicate copies of the same modules that existed in multiple locations
(Achaea/Yso/*, Achaea/Yso/Yso/*, Achaea/Yso/modules/*, Achaea/Yso/modules/Yso/*, etc.)
Canonical code is now under:
  Achaea/Yso/modules/Yso/...

Old wrapper-style requires still exist as compatibility shims in the committed tree:
  require("AffTracker"), require("LimbPressure"), require("Core.state"), require("Combat.offense_core"), etc.
New work should prefer canonical Yso modules directly, but the shims remain part of the current hybrid runtime.

Integration remains:
  require("Integration.mudlet")  -> Achaea/Yso/modules/Integration/mudlet.lua

Keep only ONE copy of each module on disk to prevent Windows case/overwrite issues.
